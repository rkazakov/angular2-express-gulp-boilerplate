# Agular 2 Express Gulp boilerplate
> Basic boilerplate project with Angular 2, Express, Gulp, and SASS

## Modules

- express
- angular2
- systemjs
- es6-shim
- reflect-metadata
- rxjs
- zone.js
- locallydb

## Development modules

- autoprefixer
- browser-sync
- cssnano
- concurrently
- gulp
- gulp-ext-replace
- gulp-imagemin
- gulp-concat
- gulp-nodemon
- gulp-rename
- gulp-postcss
- gulp-sourcemaps
- gulp-typescript
- gulp-uglify
- lite-server
- postcss
- postcss-scss
- precss

## Installation

```sh
npm install
```

## Run

```sh
npm start
```
